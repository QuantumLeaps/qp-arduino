Downloading & Installing QP/C++â„¢

The most recommended way of obtaining QP/C++™ is by downloading the QP-bundle™, which includes QP/++C™ as well as other QP frameworks and also the QM™ modeling tool and the QTools™ collection. The main advantage of obtaining QP/C++™ bundled together like that is that you get all components, tools and examples ready to go.

Note
QP-bundle™ is the most recommended way of downloading and installing QP/C++™. However, if you are allergic to installers and GUIs or don't have administrator privileges you can also download and install QP/C™ separately as described in the Application Note: "Getting Started with QP/C++" below.

Getting Started with QP/C++ App Note

The Quantum Leaps Application Note Getting Started with QP/C++ provides step-by-step instructions on how to download, install, and get started with QP/C++ quickly.
