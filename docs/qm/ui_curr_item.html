Current Item

In QM™ you always work with just one model item at a time, which is called the Current Item. All views of the model, such as the Model Explorer, the Property Editor, and the active Diagrams, highlight the same Current Item, as illustrated below:

The Current Item
Note
The Current Item is typically also the Selected Item in the active diagram. However, it is also possible to select multiple items in a diagram or, by toggling selections, to end up de-selecting the Current Item (see section Selecting Diagram Shapes). Therefore, the Selected Items in a diagram should not be confused with the Current Item, which is always highlighted in the Model Explorer and shown in the Property Editor.
