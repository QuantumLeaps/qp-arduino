Customizing QM GUIQM Model

The following sections describe how to perform basic modeling tasks in QM™:

  • QM Model explains the hierarchical structure of the QM™ model
  • Fully-Qualified Names explains how to unambiguously reference QM™ model items
  • Working with Packages explains QM™ packages for grouping other elements
  • Working with Classes explains QM™ classes
  • Working with Attributes explains class attributes and free attributes
  • Working with Class Operations explains class operations
  • Working with Free Operations explains free operations
  • Working with Events explains the event classes and event signals
  • Working with Diagrams explains the common operations on diagrams, such as: panning, zooming, selecting shapes, the grid, resizing and exporting diagrams
  • Working with Code Editors explains how various editors work in QM™
  • Search/Replace explains how you can search and replace text in QM™
  • Cut/Copy/Paste explains how to copy/cut and paste model items (even between different instances of QM™)
  • Undo/Redo explains how the feature works as implemented in QM™
  • Locking Model Items explains the concepts of protecting QM™ model items from inadvertent changes (locking)
